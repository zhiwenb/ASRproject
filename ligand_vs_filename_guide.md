# Filename Guide

This file lists the main renamed project files.

## Core scripts

- `ligand_vs_baselines.py`: main baseline active-learning pipeline
- `ligand_vs_supplemental_analyses.py`: early-round, redundancy, seed-size, and second-dataset analyses
- `ligand_vs_deep_dive_analysis.py`: budget sensitivity, calibration, and BBBP failure analysis
- `ligand_vs_custom_strategy_search.py`: custom hit-seeking acquisition experiments
- `ligand_vs_mlp_extension.py`: lightweight MLP extension experiments
- `ligand_vs_mlp_models.py`: helper models for the MLP extension
